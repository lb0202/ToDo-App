== README

This repo is made up of the resulting files of the ToDo App Tutorial- Part I by Lazarus Lazaridis 
(http://arubystory.blogspot.co.uk/2013/12/tutorial-creating-simple-todo.html). They've been uploaded for my own use 
as a way to get used to github and are not intended for further work.



Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
